# Türkçe Programlama Dili

Türkçe komutlarla yazılan basit ama tam çalışan bir programlama dili.

## Kurulum

```bash
python turkce.py program.tr
```

## Temel Komutlar

- `yaz "metin"` - Ekrana yazı yazdır
- `oku degisken` - Kullanıcıdan giriş al
- `x = 5` - Değişken ata
- `eğer x > 5 ise ... bitir` - Koşul
- `tekrarla 10 kez ... bitir` - Döngü
- `fonk adı() ... bitir` - Fonksiyon tanımla
- `çağır adı()` - Fonksiyon çağır

## Örnek Program

```turkce
yaz "Merhaba Dünya!"
x = 10
eğer x > 5 ise
  yaz "x büyüktür 5"
bitir
```
